Páginas Estáticas y Dinámicas

¿Que son las paginas estatucas?

Las páginas estáticas son aquellas cuyo contenido no cambia a menos que el desarrollador modifique el código directamente. Se crean con HTML y CSS, y son ideales para sitios simples como portafolios o páginas informativas.

¿Qué son las páginas dinámicas?

Las páginas dinámicas generan su contenido en tiempo real, normalmente utilizando lenguajes como PHP, JavaScript o frameworks modernos. Se conectan a bases de datos y permiten interacción con el usuario, como en tiendas en línea o redes sociales.

Comparación entre páginas estáticas y dinámicas

Característica Estáticas Dinámicas
Contenido Fijo Cambia según el usuario o la base de datos
Tecnologías HTML, CSS PHP, JavaScript, SQL, frameworks
Velocidad Muy rápida Depende de la carga del servidor
Ejemplos Portafolios, páginas informativas Redes sociales, tiendas en línea
